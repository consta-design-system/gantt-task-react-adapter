## Установите пакет и зависимости

`yarn add @consta/uikit @consta/gantt-task-react-adapter` или `npm install @consta/uikit @consta/gantt-task-react-adapter`

## Подключите тему

Тему обязательно нужно подключать в корне приложения, так, чтобы все компоненты `@consta/gantt-task-react-adapter` были вложены в компонент [`Theme`](##LIBS.LIB.STAND/lib:uikit/stand:components-theme-stable).

## Создайте стилизованную диаграмму

Используйте диаграмму [gantt-task-react](https://github.com/MaTeMaTuK/gantt-task-react). Подключите адаптер [useGanttTaskReactAdapter](##LIBS.LIB.STAND/lib:gantt-task-react-adapter/stand:hooks-usegantttaskreactadapter-stable), чтобы добавить стили и значения по умолчанию для некоторых свойств.
